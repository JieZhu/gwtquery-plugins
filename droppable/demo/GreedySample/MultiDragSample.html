All draggables having the same scope and the options multiDrag set to true can be selected and dragged together. To select a "selectable" draggable, just press the ctrl or cmd key and click on it.
The groupingMode option allows to rearrange the dragging display helper when the drag start. This way, you can decide to align the drag helper horizontally or vertically or both.
The select option allows to select programmatically other draggable elements that will drag in the same time that the initial draggable (i.e. the draggrable that initiate the drag operation)
